# Defender-Game
Rendition of the space game defender

This is a simple C++ game built using the SFML library.

## Features
- Dynamic gameplay with interactive features.
- Modular source code for easy customization.

## Requirements
- SFML library
- C++ compiler (e.g., g++ or clang++)
- Windows OS (tested)
